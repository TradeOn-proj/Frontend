1. ChatTest 폴더 위치에 가상환경 생성

명령어 : ChatTest\ 에서 python -m venv (가상환경이름)

2. 가상환경 활성화

명령어 : ChatTest\(가상환경이름)\Scipts\ 에서 activate

정상작동 시 프롬프트 앞에 가상환경이름 뜸

3. 필요 라이브러리 설치

명령어 : ChatTest\app\에서 pip install -r requirements.txt

4. 메인코드 실행

명령어 : ChatTest\ 에서 python -m app.main

5. 브라우저 실행 후 http://localhost:5000/chat/(방 이름) 들어가기
